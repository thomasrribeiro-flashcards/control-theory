## Card Format

Cards use the [hashcards](https://github.com/kersh1337228/hashcards) format:

```
Q: Question
A: Answer

C: Text with [cloze deletion] inline
```

Problem cards (`P:`/`S:`) walk through IDENTIFY / PLAN / EXECUTE / EVALUATE.
Card quality rules live in [CARD_STANDARD.md](../../CARD_STANDARD.md).

## Usage

Add this repository to [flashcards](https://github.com/thomasrribeiro/flashcards) while logged in with your GitHub account.
